## HTML Element

### Block vs Inline Elements
#### Block Level Elements
- 블록 레벨 요소 이전과 이후 요소 사이에 줄바꿈이 일어난다.
- 인라인 요소에 중첩될 수 없다.
- 다른 블록 레벨 요소에는 중첩될 수 있다.
```
<div>, <table>, <h1> ~ <h6>, <p>, <form>, <ul>, <ol>, <li>, <dl>, <dt>, <dd>, <pre>, <blockquote> 등
```

#### Inline Elements
- 필요한 너비만큼만 차지
```
<span>, <a>, <br>, <em>, <strong>, <input>, <label>, <img>
```
